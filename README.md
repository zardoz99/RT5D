# Repository Template – CI/CD & Release Infrastructure

This repository is an **intentional, minimal template** that provides **GitHub Actions infrastructure only**.

## What This Template Provides
- ✔ **Reliability Guardrails**: `.gitattributes` & `Directory.Build.props` pre-configured.
- ✔ **Build Workflow**: Includes automated testing and NuGet caching.
- ✔ **Release Workflow**: Automated ZIP packaging and SHA-256 checksums.
- ✔ **Standard Ignore**: Clean `.gitignore` for VS/.NET.

## How to Use
1. Click **“Use this template”**.
2. Clone your new repository.
3. Add your source code to the `src/` directory.
